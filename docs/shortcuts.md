# Keyboard Shortcuts

Use `Cmd` instead of `Ctrl` on macOS.

| Shortcut | Action |
| --- | --- |
| `Ctrl+F` | Focus search |
| `Ctrl+,` | Open settings |
| `Ctrl+R` | Refresh library |
| `Ctrl+N` | Create item |
| `Ctrl+S` | Save current file while editing |
| `Ctrl+Shift+S` | Save all files while editing |
| `F2` | Rename selected item |
| `Delete` | Delete selected item |
| `Shift+?` | Show shortcuts |
