README Utility

Generate an import button that opens Skillful with the repo prefilled.

Paste a public GitHub repository, optionally target a branch or subdirectory, and copy the finished Markdown snippet into your README. The link uses Skillful’s GitHub-safe import redirect and opens the desktop app with the import dialog prefilled.

Inputs

Build the import button

Use the repository owner/name, then add optional path and ref values only if the library lives below the repo root or you want a non-default branch or tag.
